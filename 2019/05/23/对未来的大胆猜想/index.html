对未来的大胆猜想

May 23, 2019
  • 中国能打过美国么？先说答案，不能。中国有物质代偿，简单地说，中国没了美国，粮食供给上养活不了自己，中国的6亿农民生产效率低，只能通过外汇去购买美国300万农民生产的粮食来弥补效率上的差距，在经济学上叫比较优势。
  • 通胀大杀器。为什么会有通胀，最直接的原因就是货币多了，社会生产的物质与服务不能消化多印的货币，多出的货币只能进入存量市场，从而推高了物价。如果效率一直在提高，那什么问题都没有，社会一片欣欣向荣，例如二战后。问题是效率会一直提高么，科技会一直进步么，货币会少印么，所以有了经济周期。企业、个人在经济向好的时候，都会举债多消费，多出来的货币会促进经济进一步复苏，企业个人收入增长，形势一片大好，直到供给与需求达到了平衡，供给多了，需求停滞，那问题就来了，企业盈利减少，债务却没有减少，偿债能力大减，企业就会破产，从而供给减少，物价上升，政府开始收紧货币，市场上流通的货币少了，货币购买力增强，物价下跌，居民收入减少，需求也减少，供给与需求重新达到平衡，债务清偿后，政府借贷增加，开启下一个经济周期。
  • 现阶段怎么抗通胀？钱多的买日元，美债，黄金，股票，钱少的先考虑怎么能活下去才是正途，等到资产价格大幅回落后，便宜资产可以随便买，可以购买店铺，做个买卖、收租金之类，现金牛。股市是经济的晴雨表，股市必先于经济好转，买入指数基金，随便指数基金都可以，如50ETF,300ETF,500ETF,创业板ETF，对于不懂股票的人来说，这是最好的方式了，可以实现超额收益。
  • 滞涨来了。目前的局势是既不是通胀也不是通缩，物价上涨，但是并不像之前经济过热导致的通胀一般，经济停滞，生意并不好做。因为本质是消费萎缩，将会导致通缩，但是通过货币超发和供给侧改革消除产能来反通缩造成的滞涨。
  • 房子会保值么。那你直接问水泥钢筋能保值么，房子跟人口无关，只跟货币有关，目前M2达到184万亿，大部分都是房贷贷出来的，房子不是蓄水池，是放水池，卖一套房就往社会投放了货币，货币多了，自然引发通胀，货币贬值，那你还要买房释放货币，进一步增加贬值，这是从货币上来说的，本身就是悖论。从供需上来说，需求减少，哪有不跌的道理。说说中国，居民消费已经低迷，谁会去买过剩的房子？？，除了刚需，刚需本身就是伪，纯刚需的早就买了。去年8月份消费额增速破8%的时候，10月份房子出现拐点，19年3月来了一波反弹，是因为信贷投放空前大，并没有突破前高，这意味着下降趋势形成了。
  • 原证监会主席刘shiyu自首，这位仁兄真是”人才”啊，他在任期间：1.799家企业上市，尤其是他的家乡江苏，很多企业不达上市要求上市，散户亏损严重，A股市值不升反降。2.开辟了沪港通，给窃国资本出逃开辟了道路。3.互联网金融，也就是后来的P2P,受害群众3.5亿人，被骗金额数万亿人民币，无数家庭倾家荡产，自杀。4.报告中说的，防范金融性风险，很多就是这位搞出来的。
  • P2P爆掉后，你以为结束了么，这才刚开始，债务不会凭空消失，大部分国家都不会减少开支努力还债，例如美国。中国也不会，所以债务只能违约，谁是受害者呢，当然是净资产为正的人了。再次提醒银行理财，尤其是中小银行、地方性银行。
  • 目前持仓
  1. 北方稀土吃涨停后卖出，买是为国接盘，卖是良心不安。中国是稀土最大储藏国，开采和冶炼极其污染环境，这十几年来一直在贱卖，没有贱卖，美国一直低价购买，美国也有稀土矿，早在N年前就下令停止开采。国土资源就这样流失了，后代子孙只能以高价再从美国买回。
  2. 50ETF,在周二反弹卖掉了大部分，每跌10%，会加一次仓，只做为A股是否触底的风向标参考。
  3. 黄金ETF，做网格交易，在没突破两年前1340美元之前，不会大量买入，目前仓位较轻。
  4. 山东黄金，还有下行风险，每跌5%，加仓，不怕跌，只怕筹码没吸够。
